Features

Everything on one tiny screen

DeskBuddy packs weather, clocks, timers, sensors and animated expressions into a 128×64 OLED, configured entirely from any browser on your network.

😊
Animated expressions
16 moods with spring-physics eye animations, blink, saccade and breathing. Auto-mood based on weather.
⛅
Live weather
OpenWeatherMap integration. Temperature, humidity, sunrise/sunset. 3-day forecast page included.
🕐
Clock & time
NTP-synced clock with IANA timezone support and one-click auto-detect by IP. Always accurate.
⏰
Timers & stopwatch
Up to 16 timers: daily, specific date/time, or countdown. Melody + vibration on expiry.
🌡
Indoor sensor
Optional BME280 on the same I2C bus. Temperature, humidity and pressure on a dedicated page.
🌐
Full web UI
Built-in web server. Configure weather, pins, page order, timers from any browser — no app needed.
📶
Network page
Shows SSID, IP and the web UI URL on the display so you always know how to reach it.
☀
OTA updates
ArduinoOTA built in. Update firmware wirelessly from PlatformIO without unplugging the device.

Hardware

What you need

All components are available on AliExpress for a few dollars total. The BME280 sensor and vibration motor are optional add-ons.

Required
ESP32-C3 Super Mini
The main controller. Any ESP32-C3 variant works.
Required
OLED 0.96" SSD1306
128×64 I2C display. Also compatible with 1.3" SH1106.
Required
Capacitive touch sensor
TTP223 module. Page navigation and controls.
Optional
Vibration motor
Haptic feedback on timer expiry and touch events.
Optional
Passive buzzer
Plays melodies and beeps. Share GND with motor.
Optional
BME280 sensor
Temp, humidity, pressure. Shares I2C bus with OLED.

Install

Flash directly from your browser

No drivers, no IDE, no command line. Connect your ESP32-C3 via USB and click the button below. Works on Chrome and Edge only.

1

Connect the ESP32-C3

Use a data USB cable — charge-only cables won't work. The device should appear as a serial port (COMx on Windows, /dev/ttyUSB0 on Linux).

2

Click "Install DeskBuddy"

A browser dialog will ask you to select a serial port. Choose the ESP32-C3 and click Connect.

3

Wait for the flash to complete

The installer erases the chip and writes the firmware (~1.3 MB). About 30–60 seconds. Do not disconnect.

4

Done — continue to First Boot

The device reboots automatically. Follow the steps below to configure WiFi and get started.

First boot

Getting started after flash

After the firmware is installed, follow these steps to connect DeskBuddy to your network and configure it.

1
Connect to the WiFi config portal The display shows "WiFi" and the device broadcasts an access point called YetiWifiConfig. Connect to it from your phone or laptop.
2
Enter your WiFi credentials A captive portal opens automatically. Select your home network, enter the password, and save. The device reconnects and shows its IP address.
3
Open the web UI Navigate to the IP shown on the display (e.g. http://192.168.1.42) in any browser on the same network.
4
Set your OpenWeatherMap API key Go to Settings → Weather. A free API key from openweathermap.org is required for the weather and forecast pages.
☁ How to get a free API key
1
Go to openweathermap.org/users/sign_up and create a free account.
2
After login, open My Profile → API keys. A default key is already generated — copy it.
3
Paste the key into the DeskBuddy web UI under Settings → Weather → API key and save.
4
New keys can take up to 2 hours to activate. If weather shows "No API key", wait a bit and refresh.
Free The free tier (1 000 calls/day) is more than enough — DeskBuddy fetches every 10 minutes.
5
Set your timezone Use the "Auto-detect" button in Settings → Location to automatically set your city, coordinates and IANA timezone by IP geolocation.
6
Customize and enjoy Configure page order, enable/disable pages, set auto-dim hours, add timers and explore all 16 moods from the web UI.
